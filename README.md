# archaeo 🚧

archaeo is an AI-powered developer tool designed to help engineers understand unfamiliar codebases faster.

The goal is simple:

> Understand any codebase in minutes, not days.

## Current Status

⚠️ Under Active Development

This project is currently being built and is not production-ready.

## Planned Features

* AI Repository Summary
* Architecture Analysis
* Guided Learning Paths
* AI Mentor Chat
* File Importance Detection
* Repository Exploration Tools

## Tech Stack

### Frontend

* React
* Vite
* Tailwind CSS

### Backend

* FastAPI
* SQLAlchemy
* PostgreSQL

### AI

* OpenRouter
* GPT OSS Models

## Vision

Software Archaeologist focuses on codebase comprehension rather than file browsing.

Instead of asking:

* Where should I start?
* What does this repository do?
* Which files are important?

The platform aims to answer those questions automatically.
